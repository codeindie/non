Yae is focusing on simplicity and transparency. Because of that it uses MoonScript as its scripting language. To add more transparency, Yae is also written in MoonScript, so you can have full understanding of what is engine doing by simply looking and source code you can understand.

Yae is thinking about developers in first place. To use Yae, only requirement is to have LuaRocks and JDK installed. No more tons of different tools just to package your project for other platforms. You can make games for Windows on Linux, for Linux on Windows, for Android on Mac and so on. You got the idea. Only limitation is that you cannot develop for iOS from other platform than Mac because of Apple limitations.

Yae is free and open source, and this will never change. Also, to give developers full freedom, it is licensed under the MIT license.

with yae
  name = .project.name
  music = .audio.newSource("music.ogg")
  image = .graphics.newImage("image.png")

  .load = ->
    .audio.play(music)

  .draw = ->
    .graphics.print("Hello #{name}")
    .graphics.draw(image, 10, 30)

  .quit = ->
    print "Bye #{name}"

Open Source

Yae is available under a MIT license. Visit our GitHub repository to monitor progress, report issues and even contribute with your own code!

Cross-platform

Write your game or application once, deploy on Windows, Mac, Linux, Android, iOS and Ouya.

Scripting

By leveraging latest MoonScript and using it as both scripting language and engine language, users can enjoy super fast game development process and transparent engine source code.

Powerful CLI

Yae has powerful, yet very simple to use command line interface. From it, you can create new projects, test them and release them for your desired end platform.

Simple API

API is inspired by LÖVE. That means it is super simple and easy to use. Also, you can convert your LÖVE projects to Yae without any troubles.

Easy installation

Thanks to LuaRocks, you can have full Yae installation completed in less than 5 minutes. Just see our Getting started guide and try it by yourself.

Security

Thanks to triple compilation, noone ever will be able to decompile your source code. Also, to protect your assets, you can additionally sign them. You can read more about it here.

Graphical IDE

Official Yae plugin for Sublime Text allows you to develop your game in great IDE (integrated development enviroment) with build system support and autocompletions for entire Yae API.
