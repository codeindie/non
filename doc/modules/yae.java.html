require (name)
Bind Java object to Lua

Parameters:

  • name string Java object name

Returns:

    Object a Java object converted to Lua

Usage:

     System = java.bind "java.lang.System"
 print System\currentTimeMillis!

Create new instance of specified Java object

Parameters:

  • name string Java object name
  • ... constructor parameters

Returns:

    Object an instance of specified Java class

Usage:

 print scanner\next!
