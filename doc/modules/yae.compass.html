Module yae.compass

User input from compass (works only on mobile devices).

Some Android devices and iOS devices have an integrated magnetic field sensor that provides information on how the device is oriented with respect to the magnetic north pole.

Note = The compass is currently not available on iOS devices since there is no implementation in the RoboVM - backend yet.

Functions

getAzimuth () Get the angle of the device's orientation around the z-axis.
getPitch () Get the angle of the device's orientation around the x-axis.
getRoll () Get the angle of the device's orientation around the y-axis.
getOrientation () Get current compass azimuth, pitch and roll
isAvailable () Checks if compass is available on current device


Functions

getAzimuth ()
Get the angle of the device's orientation around the z-axis. The positive z-axis points towards the earths center

Returns:

    number azimuth

Usage:

    azimuth = yae.compass.getAzimuth!
getPitch ()
Get the angle of the device's orientation around the x-axis. The positive x-axis roughly points to the west and is orthogonal to the z- and y-axis

Returns:

    number pitch

Usage:

    pitch = yae.compass.getPitch!
getRoll ()
Get the angle of the device's orientation around the y-axis. The positive y-axis points toward the magnetic north pole of the earth while remaining orthogonal to the other two axes.

Returns:

    number roll

Usage:

    roll = yae.compass.getRoll!
getOrientation ()
Get current compass azimuth, pitch and roll

Returns:

  1. number azimuth
  2. number pitch
  3. number roll

Usage:

    azimuth, pitch, roll = yae.compass.getOrientation!
isAvailable ()
Checks if compass is available on current device

Returns:

    bool True if compass is available

Usage:

    available = yae.compass.isAvailable!
