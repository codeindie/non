Module yae.timer

Provides high-resolution timing functionality.

Functions

getDelta () Returns the time between the last two frames.
getFPS () Returns the current frames per second.
getTime () Returns the value of a timer with an unspecified starting time.
sleep (seconds) Pauses the current thread for the specified amount of time.


Functions

getDelta ()
Returns the time between the last two frames.

Returns:

    number The time passed (in seconds).

Usage:

    dt = yae.timer.getDelta!
getFPS ()
Returns the current frames per second.

Returns:

    number The current FPS.

Usage:

    fps = yae.timer.getFPS!
getTime ()
Returns the value of a timer with an unspecified starting time. This function should only be used to calculate differences between points in time, as the starting time of the timer is unknown.

Returns:

    number The time in seconds.

Usage:

    time = yae.timer.getTime!
sleep (seconds)
Pauses the current thread for the specified amount of time.

Parameters:

  • seconds number Seconds to sleep for.

Usage:

    yae.timer.sleep seconds
